# Workflow Capture Align Commit

## Capture

Record fragmented ideas in `03_Sessions/Phase_*` as session notes. Do not edit formal documents during capture.

## Align

After two to three meaningful discussions, check whether the evolving understanding changes:

- goals,
- boundaries,
- assumptions,
- architecture,
- experiments, or
- active tasks.

If yes, prepare a distillation note.

## Commit

When consensus is clear, create a distillation in `03_Sessions/Distillations/` and then update the relevant formal module. Update the manifest if the project state changed.
