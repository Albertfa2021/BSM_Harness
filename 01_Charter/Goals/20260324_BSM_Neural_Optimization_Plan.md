# 科研计划方案（修订版）

## 一、题目建议

面向 ILD–ITD 联合约束的 BSM 系数神经优化方法研究

也可以写得更完整一点：

基于物理渲染前端的 BSM 系数神经优化：面向双耳 ILD–ITD 空间线索保持的联合约束方法

这个题目比原来的“残差校正”更准确，因为你现在已经明确：网络工作在系数域，输出的是对初始 BSM 系数的修正量，核心前端仍然是 BSM 渲染。 这和 BSM-iMagLS 的“DNN-based optimization framework”是同一条技术路线，只是你要把目标从“以 ILD 为主”扩展成“ILD + ITD + 信号保真”的联合优化。

## 二、研究背景与问题提出

头戴式麦克风阵列的双耳重放中，BSM 及其改进方法已经能在任意阵列几何下实现双耳渲染，其中 BSM-iMagLS 进一步通过显式引入 ILD 约束，显著改善了 ILD 误差，并在不同阵列和 head rotation compensation 场景下表现出较好的空间鲁棒性。尤其在头部旋转补偿实验中，作者明确采用了“预计算各朝向 BSM 系数 + 按头动选择系数”的框架，说明系数域优化本身就是该问题的自然形式。

但现有 BSM-iMagLS 仍主要围绕 ILD 与幅度匹配展开。另一方面，TSE 那篇文章说明：在双耳任务中，仅有信号级损失不足以充分保持空间线索，引入显式空间损失后，ITD 等空间指标还能继续改善；其提出的 ITD loss 通过 GCC-PHAT 互相关的 MSE 作为可微代理，证明了 ITD 约束是可以被纳入可训练框架的。

因此，本课题拟在 BSM 的物理渲染前端上，构建一个神经网络辅助的 BSM 系数修正框架：网络不直接输出双耳信号，而是输出对初始 BSM 系数的修正量；修正后的系数经 BSM 渲染得到双耳输出，再在输出端联合计算 ILD、ITD 和信号保真损失，从而实现对双耳空间线索的联合优化。

## 三、核心研究问题

本课题围绕以下三个问题展开。

### 1. 系数域神经优化是否可行

即：在不放弃 BSM 物理前端的前提下，神经网络能否学习到对初始 BSM 系数的有效修正，使渲染后的双耳输出在 ILD、ITD 和整体保真度上优于传统 BSM 及 BSM-iMagLS。你原始方案已经强调“问题不要做大而全，而是聚焦头部旋转与若干主要失配因素”，这个收缩方向仍然保留。

### 2. 哪种 ITD 约束更适合 BSM 系数优化

本课题不把 ITD loss 固定死，而是把它作为重要的子实验变量。计划至少比较两类 ITD 约束：

- **方案 A**：方向响应上的时域互相关 ITD
- **方案 B**：低频带上的 interaural group delay / phase slope ITD

同时保留 TSE 风格的 GCC-PHAT 工程代理思路作为实现参考。TSE 已证明“argmax 不可微时，可用互相关函数 MSE 替代”这一训练思路可行。

### 3. 空间信息层级损失与信号层级损失如何协同

你已经明确这三类损失的层级关系：

- ILD / ITD：空间信息层级损失
- MSE：信号层级损失

这一定义非常重要。它保证论文叙事不会变成普通的“重建网络”，而是明确强调：MSE 负责保真和训练稳定，ILD 与 ITD 负责双耳空间线索优化。

## 四、总体技术路线

整体结构可写成下面这条链路：

`初始 BSM 系数 + 条件输入 → 神经网络输出系数修正量 → 修正后 BSM 系数 → BSM 渲染双耳输出 → 计算 ILD / ITD / MSE 联合损失`

这里最关键的结构性原则有三条。

### 1. 网络只输出修正量，不从零预测系数

这符合你刚刚确认的设定，也更符合“神经优化”而不是“端到端替代”的定位。相比直接预测完整系数，输出修正量更稳，也更容易体现“在已有物理解附近做细调”。

### 2. 前端仍是 BSM 物理渲染

网络不是直接学习 binaural rendering，而是学习如何修正 BSM 系数，使物理渲染输出更接近目标。这一点必须作为全篇的核心定位。

### 3. ITD 设计保持可调

ITD 部分不在开题阶段锁死为单一公式，而是设计成可切换的子实验模块，用于比较不同 ITD 定义对最终空间效果的影响。

## 五、输入输出设计

### 1. 输入设计

网络输入建议分为三部分。

**（1）初始 BSM 相关信息**
包括初始 BSM 系数本身，或其等价频域表示。这是网络进行“系数修正”的直接起点。

**（2）声学条件信息**
可包括阵列传递函数相关特征、目标方向相关信息、频率索引等。因为 BSM 本身依赖阵列与目标方向关系，系数修正不能只看系数数值本身。

**（3）姿态与场景条件**
至少包括 yaw，可扩展加入 yaw rate。你原方案中把头部旋转列为重点失配因素，这一点应继续保留；若一期不想做太宽，可把 DRR 等条件作为可选辅助输入，而不是主输入。

### 2. 输出设计

网络输出为：
对初始 BSM 系数的修正量 Δw

最终用于渲染的系数为：
w_new = w_init + αΔw

其中 α 可以作为可调门控参数，在子实验中研究“修正强度”对结果的影响。你提到“希望这是可调的，作为子实验分别探究区别”，这里正好可以显式纳入方案。这样不仅有助于训练稳定，也有助于后续写论文时做出“修正量大小—性能变化”的分析图。

## 六、损失函数设计

总损失建议写成：
L = λ_ILD * L_ILD + λ_ITD * L_ITD + λ_MSE * L_MSE + λ_reg * L_reg

其中前三项对应你已经确认的三层目标，最后一项用于限制修正过大或过于不平滑。

### 1. ILD 损失
采用 BSM-iMagLS 的感知性 ILD 思路，即基于方向相关、听觉频带化的 ILD 误差，而不是简单的整段波形左右能量比。BSM-iMagLS 已表明，这种 ILD 定义更适合双耳重放问题，并已在多个阵列上获得明显优于 BSM-MagLS 的 ILD 结果。

### 2. ITD 损失
ITD 作为论文中的重点变量，采用多方案并行设计：
- **ITD-A**：方向响应上的时域互相关 ITD
- **ITD-B**：低频带上的 interaural group delay / phase slope ITD

必要时可加入一个工程参考版本，即借鉴 TSE 的 GCC-PHAT 互相关 MSE 思路，作为对照。TSE 已说明这一类代理损失是可微且有效的，但同时也提到若引入低通限制，可能更符合感知规律。

### 3. MSE 损失
MSE 放在双耳输出信号层，作用不是主导空间优化，而是约束整体保真度，防止网络为了改善 ILD 或 ITD 而牺牲过多单耳质量。

### 4. 正则项
建议加入对修正量的惩罚，如：
- 修正量幅度约束
- 频率平滑
- 姿态相邻条件下的平滑性

其作用是防止网络绕开 BSM，学成“隐式重建器”。

## 七、实验设计

### 1. 主实验目标
第一阶段建议聚焦：
固定阵列、固定或少量 HRTF、仿真场景下的头部旋转条件 BSM 系数神经优化
这个版本最可做，也最符合你前面关于可行性的判断。BSM-iMagLS 已经说明：head-tracked / head-rotation compensation 的研究完全可以在预计算系数与仿真环境中开展。

### 2. 对比方法
建议基线固定为：
- 原始 BSM
- BSM-MagLS
- BSM-iMagLS
- 你的方法：BSM + ILD + ITD + MSE

同时在你的方法内部做子实验：
- 仅 ILD + MSE
- ILD + ITD-A + MSE
- ILD + ITD-B + MSE
- 不同修正强度门控 α
- 是否加入姿态条件
- 是否加入额外正则项
这样既有对外对比，也有对内消融。

### 3. 数据设计
继续沿用你原方案中“第一版不求大而全”的原则。数据重点覆盖：
- 静态头向
- 缓慢旋转
- 中等速度旋转
- 不同 yaw 范围
- 若干典型方位
- 若干房间 / 混响条件
DRR 可以保留，但建议在第一阶段降级为次要变量，不要与 head rotation 并列为同等主任务，否则实验矩阵会迅速膨胀。你原方案中的这个风险判断是对的。

### 4. 评价指标
评价建议分成三层。
**（1）空间线索指标**
- ILD error
- ITD error
- 可选的 IPD / phase-related 指标

**（2）信号保真指标**
- MSE
- NMSE
- STFT / magnitude 类误差

**（3）动态鲁棒性指标**
- 随 yaw 变化的误差曲线
- 随 yaw rate 变化的误差曲线
- 不同旋转条件下的稳定性比较
其中第三类很重要，因为你的问题不是纯静态双耳渲染，而是带头部旋转条件的空间保持问题。BSM-iMagLS 在 head rotation compensation 下也采用了类似的动态分析思路。

## 八、理论可行性分析

本课题在理论上是可行的，主要依据有三点。
第一，BSM-iMagLS 已证明：在 BSM 系数优化框架中显式引入感知性空间约束是有效的，尤其 ILD 优化在头戴阵列场景中是有明显收益的。
第二，TSE 已证明：ITD 可以通过可微代理形式纳入神经网络训练目标，因此把 ITD 从传统评价量转为训练目标是合理的。
第三，你的方案不是让网络替代物理渲染，而是在物理解附近学习修正量，因此问题难度低于端到端直接建模，理论上更容易收敛，也更容易解释。

理论上的主要风险在于：加入 ITD 后，ILD、ITD 与 MSE 之间可能出现目标冲突；因此你把 ITD 设计成可调子实验是对的。

## 九、工程可行性分析

第一阶段工程上也是可行的，而且不强依赖昂贵仪器。
因为该课题的一期完全可以建立在：
- 现有阵列传递函数或阵列模型
- 公共 HRTF 数据
- 仿真房间
- 离线头部旋转补偿
- 普通 GPU 训练环境
之上。BSM-iMagLS 的数值验证已经覆盖 simulated and measured array transfer functions、real-world HRTF datasets，以及 head rotation compensation 分析，这说明前期研究不必须从完整在线硬件系统起步。

真正的工程门槛不在于消声室或高端跟踪设备，而在于：
- 是否已有可复现的 BSM 实现
- 阵列模型与 HRTF 坐标系是否统一
- 数据生成链条是否稳定
- 渲染层是否可微
- ITD 方案是否数值稳定
所以工程上建议先坚持“离线仿真 + 预计算系数”的版本，不把实时系统作为一期目标。

## 十、预期创新点

这版方案的创新点可以写成四条。
**1. 系数域神经优化而非信号域后处理**
网络输出的是 BSM 系数修正量，物理渲染前端保持不变，从而兼顾可解释性与学习能力。

**2. ILD–ITD 联合约束的 BSM 系数优化框架**
将 BSM-iMagLS 中成熟的感知性 ILD 思路，与 ITD 约束结合到同一个系数优化框架中。

**3. 多种 ITD 形式的系统比较**
把 ITD-A 与 ITD-B 作为方法变量，系统比较它们在双耳重放中的作用，而不是只机械套用已有公式。

**4. 面向头部旋转条件的动态空间保持**
强调在 head rotation / world-locked playback 场景下，评价空间线索是否随姿态变化保持稳定。

## 十一、阶段安排建议

**第一阶段**
完成 BSM 前端复现、数据生成链路、初始系数网络框架与 ILD + MSE 基线。

**第二阶段**
加入 ITD-A 与 ITD-B，完成联合训练与消融实验。

**第三阶段**
补充动态头转分析、修正强度门控实验，以及必要的可视化与论文撰写。
